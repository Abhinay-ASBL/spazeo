     - `files.length < 10`: toast.error("Upload at least 10 photos for good reconstruction quality")
     - `files.length > 30`: toast.error("Maximum 30 photos allowed") — also enforce via maxFiles on dropzone
   - Disable "Start Reconstruction" button when photo count < 10

   - Use a `useEffect` that runs when `files` changes and `selectedType === 'video'`
   - Store thumbnail as a data URL string and duration as a number in local state

   - Implement `handleRemovePhoto(index: number)` that removes the file at that index from the files array
   - Show live counter above the grid: "{files.length} of 10-30 photos" with a green CheckCircle icon when `files.length >= 10`

5. **Wire CaptureTips:**
   - `import { CaptureTips } from './CaptureTips'`
