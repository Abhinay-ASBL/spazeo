From RESEARCH.md — wall geometry algorithm (use exactly as specified):
```typescript
// 2D floor plan coordinate system → 3D Three.js:
// 2D x → 3D x
// 2D y → 3D z (NOT 3D y — that is vertical height)
// Wall height direction = 3D y-axis

// BoxGeometry per wall segment:
const dx = wall.end.x - wall.start.x
const dz = wall.end.y - wall.start.y  // 2D y maps to 3D z
const wallLength = Math.sqrt(dx * dx + dz * dz)
const angle = Math.atan2(dz, dx)
// Mesh center at midpoint, rotated -angle around Y axis

// Door openings: split wall into two boxes using sorted door centers along wall
// Each door: segment left of door, segment right of door
// Header above door opening (full width = door.width, height = ceilingHeight - doorHeight)

// Floor polygon — THREE.ShapeGeometry (uses Earcut internally):
const shape = new THREE.Shape()
shape.moveTo(polygon[0].x, polygon[0].y)  // 2D polygon from room
polygon.slice(1).forEach(pt => shape.lineTo(pt.x, pt.y))
shape.closePath()
const geometry = new THREE.ShapeGeometry(shape)
geometry.rotateX(-Math.PI / 2)  // rotate from XY plane to XZ plane (horizontal floor)
```

**Implementation — build geometry using useMemo:**
All geometry creation MUST be inside `useMemo` keyed on JSON-serialized geometry + overrides to prevent per-frame rebuilds. Dispose previous geometry on useMemo cleanup (geometry.dispose() in return function).

Wall geometry (`buildWalls` function, not exported):
- For each wall, find doors with door.wallId === wall.id
- Compute: dx = end.x - start.x, dz = end.y - start.y, wallLength = sqrt(dx²+dz²), angle = atan2(dz, dx)
- Coordinate mapping: 2D x → 3D x, 2D y → 3D z
- If no doors: one BoxGeometry(wallLength, ceilingHeight, wall.thickness). Position at midpoint (midX = (start.x+end.x)/2, y = ceilingHeight/2, midZ = (start.y+end.y)/2). rotation.y = -angle.
- If doors: sort by distance along wall. Iterate segments: for each gap between segments, create BoxGeometry for that segment length. Also create header above each door opening: BoxGeometry(door.width, ceilingHeight-doorHeight, wall.thickness) positioned at door center, y = doorHeight + (ceilingHeight-doorHeight)/2. Apply same rotation.
- Wall material: MeshStandardMaterial({ color: '#FFFFFF', roughness: 0.9, metalness: 0 })

Floor geometry (`buildFloors` function, not exported):
- For each room: build THREE.Shape from room.polygon (moveTo first point, lineTo rest, closePath)
- ShapeGeometry(shape), then geometry.rotateX(-Math.PI / 2) to lay flat on XZ plane
- Material color from FLOOR_MATERIAL_COLORS lookup using room.floorMaterial or overrides.roomOverrides[room.id]?.floorType
- MeshStandardMaterial({ color, roughness: 0.9, side: THREE.DoubleSide })
- mesh.position.y = 0

Ambient + directional lighting — add inside the component's returned JSX (not in Canvas, those go inside Canvas children):

**computeSceneBounds utility:**
```typescript
export function computeSceneBounds(geometry: FloorPlanGeometry) {
  let minX = Infinity, maxX = -Infinity, minZ = Infinity, maxZ = -Infinity
  geometry.walls.forEach(wall => {
    minX = Math.min(minX, wall.start.x, wall.end.x)
    maxX = Math.max(maxX, wall.start.x, wall.end.x)
    minZ = Math.min(minZ, wall.start.y, wall.end.y)
    maxZ = Math.max(maxZ, wall.start.y, wall.end.y)
  })
  const centerX = (minX + maxX) / 2
  const centerZ = (minZ + maxZ) / 2
  const sizeX = maxX - minX
  const sizeZ = maxZ - minZ
  return {
    center: [centerX, 0, centerZ] as [number, number, number],
    size: [sizeX, 0, sizeZ] as [number, number, number],
  }
}
```
