  <action>
**ExtractionProgress.tsx** -- 'use client' component:
- Props: `{ floorPlanId: Id<'floorPlanDetails'>, onComplete: (geometry: FloorPlanGeometry) => void, onError: () => void }`
- Uses `useQuery(api.floorPlanJobs.getByFloorPlan, { floorPlanId })` for reactive status
- Displays a centered card with:
  - Animated spinner (use Framer Motion rotate animation on a circular icon)
  - Status text that updates: "Uploading..." -> "Analyzing floor plan..." -> "Extracting rooms..." -> "Identifying dimensions..."
  - Progress dots or subtle pulse animation (no percentage -- AI extraction has no progress granularity)
- On status='completed': call onComplete with the job output (geometry data), trigger toast.success("Floor plan extracted successfully")
- On status='failed': show error message from job.error, display two buttons:
  - "Continue Editing Manually" -- calls onComplete with empty geometry skeleton `{ walls: [], rooms: [], doors: [], windows: [], fixtures: [], dimensions: { unit: 'meters', overallWidth: 0, overallHeight: 0 } }`
  - "Start Over" -- navigates back to /floor-plans/new
- Partial results handling: if job output has walls but no rooms, still call onComplete so user can manually add rooms
- Uses the notifications table: when extraction completes, the backend already creates a notification (from Plan 01). This component uses toast for the in-tab notification.

**AnimatedBuildUp.tsx** -- 'use client' component:
- Props: `{ geometry: FloorPlanGeometry, onAnimationComplete: () => void }`
- Uses react-konva Stage and Layer to draw the extracted geometry progressively
- Dynamic import with SSR disabled (react-konva needs DOM)
- Animation sequence using requestAnimationFrame + useState for stage tracking:
  - Stage 1 (0-1.2s): Draw wall lines progressively. Each wall Line appears with opacity animating 0 -> 1 using a staggered delay (wall index * 80ms). Walls drawn as thick Lines (stroke width 3-4px, color #F5F3EF).
  - Stage 2 (1.2-2.4s): Fill room polygons. Each room polygon fades in with a colored fill (use room type to pick color -- kitchen: warm amber, bedroom: soft blue, bathroom: light teal, living: warm gold, etc). Room name Text labels fade in at polygon centroid.
  - Stage 3 (2.4-3.5s): Add dimension annotations. Dimension lines appear with arrow markers and text labels showing measurements in meters.
- Walls with confidence="low" use amber (#FBBF24) stroke color instead of white
- After animation completes, call onAnimationComplete
- "Skip Animation" button in top-right corner that immediately shows full diagram and calls onAnimationComplete
- Canvas size: fill parent container, responsive. Calculate bounding box from geometry to set Stage dimensions with padding.
- Background: #12100E (Surface color)
