Create src/components/viewer/FurnitureGhost.tsx ('use client'):
- R3F component that renders the ghost preview when `ghostGlbUrl` is non-null in useFurnitureStore.
- Load the GLB using `useGLTF(glbUrl)`.
- Use `useFrame` to raycast from `pointer` (from `useThree`) against a floor plane at y=0 (THREE.Plane with normal (0,1,0)). Update the ghost group position to the intersection point, snapped to y=0.
- Render with `Clone` from drei, but override all materials: traverse the cloned scene and replace each mesh's material with a new `MeshBasicMaterial({ transparent: true, opacity: 0.5, color: '#D4A017' })`. Use a useEffect to traverse and swap materials after clone mounts.
- If raycaster misses the floor (e.g., pointing at sky), hide the ghost by setting visible=false on the group.
- The ghost does NOT respond to click events (raycast pass-through).

Create src/components/viewer/FurniturePiece.tsx ('use client'):
- R3F component for a single placed furniture item.
- Props: `item: PlacedItem`, `selected: boolean`, `transformMode: 'rotate' | 'scale'`, `onSelect: () => void`, `onTransformEnd: (position, rotation, scale) => void`.
- Load GLB using `useGLTF(item.glbUrl)`.
- Render as `<group>` with position/rotation/scale from item. Use `Clone` from drei for the model.
- When `selected` is true:
  - Show a Gold selection outline. Implement by traversing cloned meshes and adding an emissive glow (set emissive color to '#D4A017', emissiveIntensity to 0.3) on the original materials. Reset when deselected.
  - Render `<TransformControls>` from drei with `mode={transformMode}` attached to the group ref. TransformControls automatically disables OrbitControls during gizmo drag.
  - On `onObjectChange` event from TransformControls, read the group's position/rotation/scale. On `mouseUp` (drag end), call `onTransformEnd` with the new values. This avoids pushing to undo stack on every frame during drag.
- onClick on the group: call `onSelect()` with `e.stopPropagation()` to prevent deselecting.
- Do NOT allow translate mode — only rotate and scale. Items stay at their placed position (y=0 enforced). Re-positioning requires delete + re-place.

Create src/components/viewer/FurnitureLayer.tsx ('use client'):
- R3F component that orchestrates all furniture interaction inside the Canvas.
- Reads from useFurnitureStore: mode, ghostItemId, ghostGlbUrl, placedItems, selectedId, transformMode, and all action methods.
- **Rendering:**
  - If `ghostGlbUrl` is set, render `<FurnitureGhost glbUrl={ghostGlbUrl} />`
  - Map over `placedItems` and render `<FurniturePiece>` for each, passing `selected={item.instanceId === selectedId}` and `transformMode`.
- **Click-to-place logic:**
  - Listen for onClick on a transparent full-scene click catcher (an invisible plane at y=0 using `<mesh>` with transparent material). Only active when `mode === 'furnish'` AND `ghostItemId` is set.
  - On click: raycast to floor plane, get intersection point. Create a new PlacedItem with `crypto.randomUUID()` as instanceId, position from intersection (y=0), default rotation [0,0,0], default scale [1,1,1], and metadata (name, price, amazonUrl, glbUrl) from the catalog item. The catalog item data should be passed via the store or looked up. Store the item name/price/amazonUrl in the ghost state (extend ghost to carry metadata, or look up from a ref).
  - Call `addItem(newItem)` then `clearGhost()` to finalize placement.
  - Actually, the cleaner approach: when setGhostItem is called from the catalog, also store ghostMetadata (name, price, amazonUrl) in the store. Extend useFurnitureStore in Plan 01 with `ghostMetadata: { name: string; price: number; amazonUrl?: string } | null` and `setGhostItem(id, url, metadata)`.
  - NOTE TO EXECUTOR: If useFurnitureStore from Plan 01 doesn't have ghostMetadata, add it — this is within Claude's discretion per CONTEXT.md.
- **Click-to-deselect:** Clicking the floor (not a furniture piece) while no ghost is active deselects the current item (`setSelectedId(null)`).
- **Item selection:** Each FurniturePiece's onSelect sets `setSelectedId(instanceId)`.
- **Transform end:** Each FurniturePiece's onTransformEnd calls `updateTransform(instanceId, position, rotation, scale)`.
- **Keyboard shortcuts** (useEffect with document keydown listener):
  - Guard: skip if event target is input/textarea, or if mode !== 'furnish'
  - `R` key: setTransformMode('rotate')
  - `S` key: setTransformMode('scale')
  - `Delete` or `Backspace`: if selectedId, removeItem(selectedId) then setSelectedId(null)
  - `Ctrl/Cmd + Z`: e.preventDefault(), call undo()
  - `Escape`: if ghost active, clearGhost(). If item selected, setSelectedId(null).
- This component is rendered inside the R3F Canvas, alongside existing NavigationModes, SplatScene, etc.
