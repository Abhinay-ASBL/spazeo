Create src/components/viewer/FurnitureCameraController.tsx ('use client'):
- R3F component rendered inside the Canvas alongside FurnitureLayer.
- Reads `centerOnItem` and `setCenterOnItem` from useFurnitureStore.
- Uses `useThree` to get the OrbitControls ref (via `controls` from useThree, or obtain OrbitControls ref from parent via context/prop).

- This wires up the user decision: "Clicking an item in the cost list selects it in the 3D scene and centers camera on it." CostTracker (Plan 02) calls setCenterOnItem(position), and this component consumes it.
